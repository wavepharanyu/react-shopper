# React Shopper
# Available Scripts

## Frontend
### `cd frontend`
### `npm install`
### `npm run dev`

## Backend
### `cd backend`
### `npm install`
### `npm run server`

## Admin Side
### `cd admin`
### `npm install`
### `npm run dev`

* ![React][React.js]
* ![MongoDB][MongoDB]
